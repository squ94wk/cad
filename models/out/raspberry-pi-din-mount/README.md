# Raspberry Pi DIN Rail Mounts

OpenSCAD designs for mounting various Raspberry Pi models to DIN rails.

## Models

- `pi5.scad` - Raspberry Pi 5 mount
- `pi5_mirror.scad` - Raspberry Pi 5 mirrored mount
- `pi5_ssd.scad` - Raspberry Pi 5 mount with additional 2.5" SSD mounting holes
- `cm5.scad` - Compute Module 5 mount
- `ssd_hat_mount.scad` - supporting mount for Radxa Penta SATA Hat
